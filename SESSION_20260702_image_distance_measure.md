# 会话记录 - 图片两点测距工具

## 功能目标
- 选择图片并在图片上点击两个点
- 计算两点之间的像素距离
- 输入像素精度，换算实际距离

## 修改记录
- 新建 `ImageDistanceMeasure` Qt C++ 项目
- 实现 `ImageLabel` 自定义控件，支持缩放显示与坐标映射
- 实现主界面：选图、清除选点、精度输入、距离显示
- 像素精度改为计算形式：标定长度(mm) ÷ 标定像素(pixel) = 像素精度(mm/pixel)
- 图片支持放大缩小：滚轮缩放、工具栏按钮、中键拖拽平移

## 使用说明
1. 点击「选择图片」加载 bmp/jpg/png 等图片
2. **滚轮**放大/缩小图片，**中键拖拽**平移查看
3. 工具栏提供「放大」「缩小」「适应窗口」按钮
4. 在「标定长度」输入已知物体实际长度（mm）
5. 在「标定像素」输入该物体在图片上的像素长度（pixel）
6. 程序自动计算并显示「像素精度 = 标定长度 ÷ 标定像素」
7. 在图片上依次左键点击两个点，查看像素距离和实际距离（mm）
8. 点击「清除选点」可重新选点
