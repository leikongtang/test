# 会话记录 - 图片两点测距工具

## 功能目标
- 选择图片并在图片上点击两个点
- 计算两点之间的像素距离
- 输入像素精度，换算实际距离

## 修改记录
- 新建 `ImageDistanceMeasure` Qt C++ 项目
- 实现 `ImageLabel` 自定义控件，支持缩放显示与坐标映射
- 实现主界面：选图、清除选点、精度输入、距离显示

## 编译与运行
- Release 编译路径: `ImageDistanceMeasure/build-release/release/ImageDistanceMeasure.exe`
- 已验证 Release 编译通过并成功启动

## 使用说明
1. 点击「选择图片」加载 bmp/jpg/png 等图片
2. 在图片上依次左键点击两个点（自动显示红点和连线）
3. 底部输入「像素精度」（mm/px，即每像素对应的实际长度）
4. 界面自动显示：两点坐标、像素距离、实际距离（mm）
5. 点击「清除选点」可重新选点；选满两点后再点会自动重新开始
