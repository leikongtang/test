# 会话记录 - 图片两点测距工具

## 功能目标
- 选择图片并在图片上点击两个点
- 计算两点之间的像素距离
- 输入像素精度，换算实际距离

## 修改记录
- 新建 `ImageDistanceMeasure` Qt C++ 项目
- 实现 `ImageLabel` 自定义控件，支持缩放显示与坐标映射
- 实现主界面：选图、清除选点、精度输入、距离显示
- 像素精度改为计算形式：标定长度(mm) ÷ 标定像素(pixel) = 像素精度(mm/pixel)
- 图片支持放大缩小：滚轮缩放、工具栏按钮、中键拖拽平移
- 新增「移动图片」模式：左键拖拽平移；右键/中键也可直接拖拽
- 新增「隐藏点1」「隐藏点2」「隐藏线段」开关：可分别隐藏选点与连线，保留测距数据
- 图片上及底部信息栏显示两点像素坐标 (x, y) pixel
- 可选垂直辅助线：过两点各一条线，点1/点2角度独立可调（0° 为竖直）
- 开启「滚轮调点1/点2」后，在图片上滚轮调整对应辅助线角度
- 点1、点2、辅助线1、辅助线2、测距线均可自定义颜色
- 选满两点后锁定，必须「清除选点」才能重新选择

## 使用说明
1. 点击「选择图片」加载 bmp/jpg/png 等图片
2. **滚轮**放大/缩小图片
3. **移动图片**：点击工具栏按钮开启后左键拖拽；或直接用**右键/中键**拖拽平移
4. **分段隐藏**：「隐藏点1」「隐藏点2」「隐藏线段」可独立切换，再次点击恢复显示
5. 工具栏提供「放大」「缩小」「适应窗口」按钮
6. 在「标定长度」输入已知物体实际长度（mm）
7. 在「标定像素」输入该物体在图片上的像素长度（pixel）
8. 程序自动计算并显示「像素精度 = 标定长度 ÷ 标定像素」
9. 关闭移动模式后，在图片上左键点击两个点，查看像素距离和实际距离（mm）
10. 点击「清除选点」可重新选点
