# 会话记录 - 龙形桌宠 DragonDeskPet

## 功能目标
- 透明无边框置顶桌宠窗口
- 手绘卡通小龙形象，含呼吸、摆尾、扇翅、眨眼动画
- 鼠标拖拽移动，双击触发喷火/开心动作
- 右键菜单退出

## 修改记录
- 新建 `DragonDeskPet` Qt C++ 项目
- 实现透明无边框置顶桌宠窗口 `DragonWidget`
- 手绘青绿色卡通小龙：角、耳鳍、翅膀、尾巴、背脊
- 动画：呼吸缩放、上下浮动、摆尾、扇翅、随机眨眼、瞳孔微动
- 双击喷火特效；左键拖拽移动；右键菜单退出
- 修复 Qt 平台插件缺失：Release 编译后自动 windeployqt 部署
- 新增 `deploy.bat` 手动部署脚本
- **互动增强**：
  - 单击头部/身体/尾巴分别触发抚摸、揉肚、摇尾反应（爱心、腮红、对话气泡）
  - 鼠标悬停时瞳孔跟随、头部微倾
  - 右键菜单：喂食（苹果下落动画）、说句话、跟随鼠标开关
  - 闲置 40 秒自动睡觉（ZZZ），鼠标靠近或点击唤醒
  - 双击喷火并显示对话

## 使用说明
1. 编译 Release 后运行 `DragonDeskPet/build-release/release/DragonDeskPet.exe`
2. 若提示缺少 Qt platform plugin，执行 `DragonDeskPet/deploy.bat` 或重新编译（已自动部署）
3. **单击**头部/肚子/尾巴 → 不同互动反应
4. **拖拽**空白区域移动小龙
5. **双击**喷火
6. **右键**菜单：喂食 / 说句话 / 跟随鼠标 / 退出
7. 鼠标悬停时龙龙会看你；长时间不理会它会睡着
