# 会话记录 - 龙形桌宠 DragonDeskPet

## 功能目标
- 透明无边框置顶桌宠窗口
- 手绘卡通小龙形象，含呼吸、摆尾、扇翅、眨眼动画
- 鼠标拖拽移动，双击触发喷火/开心动作
- 右键菜单退出

## 修改记录
- 新建 `DragonDeskPet` Qt C++ 项目
- 实现透明无边框置顶桌宠窗口 `DragonWidget`
- 手绘青绿色卡通小龙：角、耳鳍、翅膀、尾巴、背脊
- 动画：呼吸缩放、上下浮动、摆尾、扇翅、随机眨眼、瞳孔微动
- 双击喷火特效；左键拖拽移动；右键菜单退出

## 使用说明
1. 运行 `DragonDeskPet/build-release/release/DragonDeskPet.exe`
2. **左键拖拽**移动小龙到任意位置
3. **双击**触发喷火动画
4. **右键**打开菜单，选择「退出」关闭桌宠
