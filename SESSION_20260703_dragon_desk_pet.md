# 会话记录 - 龙形桌宠 DragonDeskPet

## 功能目标
- 透明无边框置顶桌宠窗口
- 手绘卡通小龙形象，含呼吸、摆尾、扇翅、眨眼动画
- 鼠标拖拽移动，双击触发喷火/开心动作
- 右键菜单退出

## 使用说明
1. 编译 Release 后运行 `DragonDeskPet/build-release/release/DragonDeskPet.exe`
2. 若提示缺少 Qt platform plugin，执行 `DragonDeskPet/deploy.bat` 或重新编译（已自动部署）
3. **单击**头部/肚子/尾巴 → 不同互动反应
4. **拖拽**空白区域移动小龙
5. **双击**喷火
6. **右键**菜单：喂食 / 说句话 / 跟随鼠标 / 退出
7. 鼠标悬停时龙龙会看你；长时间不理会它会睡着
